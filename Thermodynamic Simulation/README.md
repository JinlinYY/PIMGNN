# 🧪 Ternary LLE Thermodynamic Simulation (UNIFAC / NRTL / COSMO-RS)

This repository is dedicated to the thermodynamic phase equilibrium calculation and simulation of multi-component complex systems (e.g., extractive distillation and liquid-liquid separation systems involving sulfolane). By combining **quantum chemistry predictions (COSMO-RS)** with **macroscopic thermodynamic models (UNIFAC & NRTL)**, it generates broad-range liquid-liquid equilibrium (LLE) data and plots phase envelopes.

The Python scripts in this project are built upon the open-source phase equilibrium library [Phasepy](https://github.com/gustavochm/phasepy), and feature an innovative custom group-injection method to significantly expand the native library's computational boundaries.

---

## 1. Quantum Chemistry Simulation (COSMOthermX) ⚛️

In scenarios where experimental data is lacking, or a microscopic understanding of intermolecular interactions is required, this project utilizes the **COSMOthermX** software based on the COSMO-RS (Conductor-like Screening Model for Real Solvents) theory.

* **Core Function:** Accurately predicts the chemical potential of molecules utilizing the screening charge density ($\sigma$-profile) derived from first-principles calculations.
* **Applications:** Predicts crucial thermodynamic properties—such as ternary Liquid-Liquid Equilibrium (LLE), Vapor-Liquid Equilibrium (VLE), and binary Excess Gibbs Free Energy—directly from molecular structures, without the need for macroscopic experimental data regression. This is particularly critical for ternary systems containing ionic liquids, salts, and complex solvents.
* **Usage:** This simulation is performed independently in the COSMOthermX software (requires importing `.cosmo` files generated by quantum chemistry software like Gaussian or Turbomole). The prediction results serve as an extremely important benchmark for initial guesses and cross-validation of parameters in macroscopic models like NRTL.

## 2. Macroscopic Thermodynamics (UNIFAC & NRTL) 💻

This repository contains a highly integrated automation Python script, **`UNIFAC-NRTL.py`**, designed for generating liquid-liquid binodal curve data for various ternary systems across temperature gradients. Currently supported systems include:

1. Water + Diethoxymethane (DEM) + p-Xylene @ 303.15 K
2. n-Heptane + Toluene + Sulfolane @ 348.15 K

### 🌟 Core Features

1. **Phasepy Extension & Custom Group Injection**
   Since the native **[`phasepy`](https://github.com/gustavochm/phasepy)** UNIFAC parameter database may lack certain specific non-standard functional groups (such as `TMSO2` for sulfolane or `THIOPHE` for thiophene), the code innovatively introduces the `build_custom_unifac_model()` function. This module bypasses the underlying initialization checks by constructing a dummy mixture, and then directly injects custom volume parameters ($R_k$), surface area parameters ($Q_k$), and group interaction parameters ($a_{mn}$) into the memory matrix, successfully enabling UNIFAC calculations for complex heterocyclic compounds.
   
2. **Seamless NRTL Parameter Comparison**
   The code framework reserves standardized matrix inputs for NRTL model interaction parameters ($g_{ij}$ and $\alpha_{ij}$). Researchers can conveniently input experimentally regressed NRTL parameters to directly compare them with UNIFAC prediction results under identical conditions.
   
3. **Broad Phase Envelope Generation via Continuation Method**
   Through the built-in `generate_broad_phase_envelope` algorithm module, the program automatically advances the scanning of the solute feed ratio starting from an extremely low concentration (0.1%). By using the converged solution of the previous point as the initial guess for the next state point, it significantly improves the convergence stability of complex nonlinear LLE equation systems, allowing for smooth scanning up to the vicinity of the Plait Point.
