# 🧪 Ternary LLE Thermodynamic Simulation (UNIFAC / NRTL / COSMO-RS)

This repository is dedicated to the thermodynamic phase equilibrium calculation and simulation of multi-component complex systems (e.g., extractive distillation and liquid-liquid separation systems involving sulfolane). It combines **quantum chemistry predictions (COSMO-RS)** with **macroscopic thermodynamic models (UNIFAC & NRTL)** to generate broad-range liquid-liquid equilibrium (LLE) data and plot phase envelopes.

The Python scripts in this project are built upon the open-source phase equilibrium library [Phasepy](https://github.com/gustavochm/phasepy), and feature a custom group-injection method to expand the library's native computational capabilities.

---

## 1. Quantum Chemistry Simulation (COSMOthermX) ⚛️

In scenarios where experimental data is lacking, or a microscopic understanding of intermolecular interactions is required, this project utilizes **COSMOthermX** based on the COSMO-RS (Conductor-like Screening Model for Real Solvents) theory.

* **Core Function:** Accurately predicts the chemical potential of molecules utilizing the screening charge density (σ-profile) derived from first-principles calculations.
* **Applications:** Predicts crucial thermodynamic properties—such as ternary Liquid-Liquid Equilibrium (LLE), Vapor-Liquid Equilibrium (VLE), and binary Excess Gibbs Free Energy—directly from molecular structures, without the need for macroscopic experimental data regression. This is particularly useful for systems containing ionic liquids, salts, and complex solvents.
* **Usage:** This simulation is performed independently in the COSMOthermX software (requires importing `.cosmo` files generated by quantum chemistry software like Gaussian or Turbomole). The prediction results serve as a vital benchmark for initial guesses and cross-validation of parameters in macroscopic models like NRTL.

## 2. Macroscopic Thermodynamics (UNIFAC & NRTL) 💻

This repository contains two main Python scripts designed for generating liquid-liquid binodal curve data for different systems:

* **`hj46.py`**: Dedicated to the phase equilibrium simulation of the **n-Heptane + Toluene + Sulfolane** system (fixed temperature scanning).
* **`heji.py`**: A multi-system batch script that generates phase diagram data across temperature gradients. Currently supports:
    1.  Water + Diethoxymethane (DEM) + p-Xylene
    2.  Octane + Thiophene + Sulfolane
    3.  n-Hexane + Ethyl acetate + Sulfolane

### 🌟 Core Features

1.  **Phasepy Extension & Custom Group Injection**
    Since the native `phasepy` UNIFAC parameter database may lack certain specific functional groups (such as `TMSO2` for sulfolane or `THIOPHE` for thiophene), this code innovatively introduces the `build_custom_unifac_model()` function. This function bypasses the underlying initialization checks by constructing a dummy mixture, and then directly injects custom volume parameters ($R_k$), surface area parameters ($Q_k$), and group interaction parameters ($a_{mn}$) into the memory matrix, successfully enabling UNIFAC calculations for complex compounds.
2.  **NRTL Parameter Comparison**
    The code reserves placeholder matrices for NRTL model interaction parameters ($g_{ij}$ and $\alpha_{ij}$). This allows users to easily compare experimental regressed parameters with UNIFAC predictions under identical conditions.
3.  **Broad Phase Envelope Generation via Continuation Method**
    Through the `generate_broad_phase_envelope` module, the code automatically scans the solute concentration starting from extremely low fractions. By using the converged result of the previous point as the initial guess for the next point, it significantly improves the convergence rate of complex LLE calculations, allowing for smooth scanning up to the vicinity of the Plait Point.
